Configura la conexión a Supabase

Para que Mia Clean guarde los datos en la nube y se vean actualizados en cualquier dispositivo, completa estos dos valores en el código, dentro de la etiqueta <script> al final del archivo:

const SUPABASE_URL = 'https://xxxxxxxx.supabase.co';
const SUPABASE_ANON_KEY = 'eyJhbGciOi...tu-anon-key...';
  1. Crea un proyecto gratis en supabase.com.
  2. Ejecuta el script crear_tablas_supabase.sql en el SQL Editor de tu proyecto.
  3. Ve a Project Settings > API y copia el "Project URL" y la "anon public key".
  4. Pégalos en las dos constantes de arriba y guarda/recarga este archivo.
